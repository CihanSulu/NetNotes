Microsoft Net Core ile artık daha hafif ve daha hızlı uygulamalar geliştirmek mümkün. 
###### Kursun sonunda:

- Asp Net Core MVC ile projeleri rahalıkla geliştirilebilecek.
    
- N Katmanlı Mimariyi detaylı bir şekilde öğrenilecek.
    
- Asp Net Core Identity ile projelerinize kullanıcı giriş sistemleri eklenebilecek.
    
- Entity Framework Core mantığına tamamen hakim olunacak.
    
- Migration ları kolaylıkla oluşturup, database tarafına ilgili değişiklikleri yansıtılabilecek.
    
- Asynchronous Programming Model mantığını anlayabilecek.
    
- AutoMapper ile entityleri dtolara kolaylıkla çevirebilecek.
    
- Visual Studio Code ile uygulama geliştirebilecek.
    
- Net Core Web Api ile servisler yazacak, yazdığınız servisleri, Mvc ile rahatlıkla tüketebilecek.
    
- JWT (Json Web Token) ile authentication sağlayabilecek ve bu işin mantığına hakim olunabilinecek.
    
- Unit Of Work ile transactionları yönetebilecek, transaction  'ın mantığını anlaşılacak.
    
- Fluent Validation ile SRP (Single Responsibility Principle) ' yi çiğnemeden, validation rule lar yazılabilecek.
    
- Fluent Api aracılığı ile, entitylerinizi configure edilebilecek.
    
- Repository Design Pattern ile kendinizi tekrar etmemeyi öğrenilecek.
    
- Solid prensiplerini detaylı bir şekilde öğrenecek, projelerinizde uygulanabilinecek.
    
- DirectoryInfo ve FileInfo classlarını ve metodlarını öğrenecek, projelerinizde uygulanabilinecek.
    
- Microservice mimarisi hakkında temel düzeyde bilgi sahibi olunacak.
