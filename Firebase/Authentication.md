Firebase #Authentication ile kullanıcı login işlemleri Firebase üzerinden kolaylıkla yönetilebilmektedir. Bunlar;

- Email/Password
- Phone
- Anonymous
- Google
- Facebook
- Play Games
- Game Center
- Apple
- Github
- Microsoft
- Twitter
- Yahoo

İle kullanılmaktadır. Bunun içini Firebase console hesabı üzerinden
- Security-> Authentication 
sayfasından erişilebilir.

##### Ek Ayarlar
Aynı zamanda Authentication yaparken mail, sms şablonları da kullanılabilmektedir.

Ek olarak günlük aynı ipden erişim sayısı, şifre politikası,smtp ayarları,recaptcha gibi ayarlarda yapılabilemktedir.
